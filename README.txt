WELCOME TO CLEAN WATER CROWSOURCING (TEMPORARY NAME FOR APP)
------------------------------------------------------------
This is an application that allows users to find clean drinking water and to report on testing results.

Thus far a user can open to the main screen of the application and has an option to login or register. If the user registers, the a screen where the user can enter his or her information will appear. Once verified, the user will be taken to the Application screen where the user will be able to logout back to the Main screen. If the user presses the Register Button, the user will be taken to a registration screen where he or she can register for an account.
